Website Game Introduction


Introduction

The game is essentially the same as the game breakout but instead of having a moving bar to block the balls (the blue balls) you have your own ball (the green ball) instead. This ball you control has its own momentum and is controlled by rockets that will nudge it in a certain direction. Like breakout you need to destroy all the bricks at the top of the screen with the blue balls while preventing the same blue balls from hitting the bottom of the screen.


Controls

Use the arrow keys to control the green ball


Inter-level Interface

THe Inter-level interface will allow you give your name and select your level (through the buttons displayed). As you complete a level a new harder level will be made available (available levels stored in browser cookies).
